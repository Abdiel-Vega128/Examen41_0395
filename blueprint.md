# Blueprint

## Overview

This is a Flutter application for a restaurant UI. It has three screens: Inicio, Pantalla2, and Pantalla3.

## Style and Design

- The app uses Material Design.
- The initial route is the Inicio screen.

## Features

- The app has three screens with named routes:
    - `/`: Inicio
    - `/pantalla2`: Pantalla2
    - `/pantalla3`: Pantalla3

## Plan

1.  Run the app on the Android emulator to verify the fix for the Android SDK issue.
2.  If the app runs successfully, the issue is resolved.
3.  If the issue persists, further investigation will be required.
